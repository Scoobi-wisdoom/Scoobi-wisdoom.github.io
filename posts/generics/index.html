Generics

August 10, 2020

Generics에 대해 알아보자. 타입 변수 or 타입 매개변수, 대입된 타입(parameterized type), 원시 타입(raw type), generic class, 참조 변수(reference variable) return type을 정해주기 위해 사용한다.

Parameterized type에는 primitive type이 올 수 없고 reference type만 올 수 있다.

Generic class의 개체를 생성할 때, 참조 변수에 지정해준 generic type과 생성자에 지정해준 generic type은 일치해야 한다.

“A generic type is a generic class or interface that is parameterized over types.”
